Список доступных возможностей:
/getip - получить IP адреса NGROK
/kill - убить процесс бота
/nircmd *команда* - выполнить nircmd команду
/say *текст* - сказать говорилкой текст
/sc - скриншот рабочего стола
/shell *команда* - выполнить cmd команду
/startngrok - запустить NGROK
/stop - завершить запущенные функции
/stopngrok - остановить NGROK
/vol *значение* - установить уровень громкости

Для администратора:
/on - включить бота
/off - выключить бота

Пример:
/say нюхаю бебру каждый день

Поддерживается воспроизведение видео, аудио, голосовых сообщений, фото, фото из вк по ссылке, видео из вк по ссылке, видеосообщений, видео из ютуба, видео из тиктока, файлов
